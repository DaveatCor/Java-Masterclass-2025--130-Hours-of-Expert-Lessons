## The Terminal Operations that return Optional
In this video, we'll be covering the final few terminal operations, all of which return an Optional.

| Return Type    | Terminal Operations                   |
|----------------|---------------------------------------|
| OptionalDouble | average()                             |
| Optional<T>    | finadAny()                            |
| Optional<T>    | finadFirst()                          |
| Optional<T>    | max(Compartor<? super T> comparator)  |
| Optional<T>    | min(Compartor<? super T> comparator)  |
| Optional<T>    | reduce(BinaryOperator<T> accumulator) |
