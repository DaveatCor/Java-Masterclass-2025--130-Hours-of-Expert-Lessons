## Devising A Teen Number Checker For Mastery Of Conditional Range Checking

```java
public class TeenNumberChecker {
    public static boolean hasTeen(int param1, int param2, int param3){
        if (param1 >= 13 && param1 <= 19){
            return true;
        }
        
        if (param2 >= 13 && param2 <= 19){
            return true;
        }
        
        if (param3 >= 13 && param3 <= 19){
            return true;
        }
        
        return false;
    }
    
    public static boolean isTeen(int param){
        if (param >= 13 && param <= 19){
            return true;
        }
        
        return false;
    }
}
```
